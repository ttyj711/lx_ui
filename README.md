# 旅途账本 - 旅游出行费用记录 APP

一款现代化的多人旅行费用记录与统计手机应用原型。

## 项目简介

旅途账本帮助用户在旅行过程中轻松记录每笔消费，智能计算多人费用分摊，并提供详细的统计分析。让旅行记账变得简单、透明、高效。

## 核心功能

### 📝 快速记账
- 创建旅程（设置日期、总预算、同行成员）
- 一键记录消费（金额、分类、地点、参与成员）
- 自动关联旅行日程（Day1、Day2...）
- 支持上传小票照片

### 👥 智能分摊
- 根据实际参与成员自动计算人均费用
- 非简单平均分摊，更公平合理
- 支持部分成员参与的消费场景

### 📊 统计分析
- 消费结构分析（餐饮、住宿、交通等）
- 每日消费趋势图表
- 人均消费排行榜
- 预算执行情况实时监控

### 📈 旅行总结
- 自动生成旅行消费报告
- 总消费、人均消费、预算执行率
- 消费类别占比可视化
- 支持导出 PDF、Excel、长图分享

## 使用说明

1. 打开 `index.html` 进入应用首页
2. 点击底部导航切换不同页面
3. 首页查看当前旅程概况和今日消费
4. 点击记账按钮快速新增消费记录
5. 统计页查看详细的消费分析数据
6. 旅行结束后在总结页生成报告
